使用generator-sogou

环境

  • 安装 node： https://nodejs.org/
  • 安装 Yeoman ：$ npm install -g yo
  • 安装 generator-sogou ：$ npm install -g generator-sogou

使用方法

  • 创建sogou浏览器插件： $ yo sogou
  • 打包sext文件并实时刷新插件：$ gulp build
  • 发布新版插件：$ gulp publish

总结实现逻辑

  • yo来生成项目目录
  • gulp实时监控code变化并刷新插件
  • gulp实现压缩去除注释js引用以及代码注释更新manifest文件
